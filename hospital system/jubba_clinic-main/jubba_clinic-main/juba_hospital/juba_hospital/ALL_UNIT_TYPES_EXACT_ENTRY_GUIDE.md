# ğŸ” UNDERSTANDING THE FIELDS

## Field-by-Field Explanation

### Unit Name
- **What:** The full name of the unit category
- **Examples:** Tablet, Syrup, Injection
- **Tip:** Use standard medical terminology

### Unit Abbreviation
- **What:** Short form (max 10 characters)
- **Examples:** Tab, Syr, Inj
- **Tip:** Keep it recognizable

### Selling Method (Dropdown)
- **countable:** For items you count (most medicines)
- **volume:** For liquids measured in ml
- **weight:** For items measured by weight (rarely used)

### Base Unit Name
- **What:** The smallest individual item
- **For Tablets:** piece
- **For Syrup:** ml
- **For Injections:** vial
- **For Ointments:** tube

### Allows Subdivision (Checkbox)
- **Check âœ“** if medicine comes in packages (strips, boxes)
- **Leave blank** if only sold individually

### Subdivision Unit
- **What:** The packaging name
- **For Tablets:** strip (tablets in blister pack)
- **For Syrup:** bottle (ml in a bottle)
- **For Injections:** box (vials in a box)

### Unit Size Label
- **What:** Describes relationship
- **Examples:**
  - "pieces per strip"
  - "ml per bottle"
  - "vials per box"

### Is Active (Checkbox)
- **Check âœ“** to make unit available
- **Uncheck** to deactivate (for old units)
